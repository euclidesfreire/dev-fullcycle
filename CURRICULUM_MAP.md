# Curriculum Map

Mapa inicial da formação. O conteúdo detalhado dos livros será planejado antes da escrita dos capítulos.

## Trilhas obrigatórias

1. Fundamentos de computação
2. Engenharia de software
3. Desenvolvimento full stack
4. Backend com Java Spring Boot
5. Backend com Node.js e TypeScript
6. Frontend com React e TypeScript
7. Bancos de dados
8. Sistemas distribuídos
9. DevOps
10. Cloud
11. Inteligência artificial aplicada
12. LLMs
13. RAG
14. Agentes de IA
15. Cybersegurança
16. Segurança em aplicações
17. Segurança em APIs
18. Segurança em IA
19. Projeto final integrado

## Regra de evolução

Cada capítulo deve declarar pré-requisitos, competências, dependências e conexão com o projeto final antes de ser aprovado.


## Livro 1 — Fundamentos de Computação e Engenharia de Software

Objetivo: formar a base conceitual necessária antes de Desenvolvimento Full Stack, IA e Cybersegurança, conectando fundamentos computacionais, engenharia de software, segurança, operação e pensamento arquitetural.

### Projeto final do Livro 1

Ao longo dos capítulos, o leitor prepara um sistema mínimo, versionado, testável, seguro por design e observável. O objetivo não é competir com os projetos das trilhas Full Stack, IA ou Cybersegurança, mas criar a base mental e técnica para entendê-los com rigor.
